# Cyber Threat Watch

A Cyber Threat Watch landing site built with Vite, React, and TanStack Start.

## Technologies

- React 19
- Vite
- TanStack React Router
- TanStack React Query
- Tailwind CSS + Tailwind Merge
- Radix UI components
- Cloudflare Workers compatible build

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173/` by default.

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Quality

```bash
npm run lint
npm run format
```

## Notes

- The repository is configured for a Vite + Cloudflare Workers deployment.
- `package-lock.json` is included to lock dependency versions.
